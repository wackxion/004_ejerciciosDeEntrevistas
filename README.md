# 004_ejerciciosDeEntrevistas
# Opción A: El clásico "FizzBuzz" (Lógica de Divisibilidad)
Es el ejercicio de lógica más famoso en las entrevistas de programación.
El Reto: Imprimir los números del 1 al 100. Pero:
Si el número es divisible por 3, imprime "Fizz".
Si es divisible por 5, imprime "Buzz".
Si es divisible por ambos (3 y 5), imprime "FizzBuzz".
¿Qué practicas?: El uso del operador módulo (%), bucles for y condicionales anidados
.
# Opción B: Verificador de Palíndromos (Lógica de Strings)
Ideal para practicar la manipulación de textos, algo muy útil para cuando proceses datos de usuarios.
El Reto: Crear un programa que reciba una palabra o frase y determine si se lee igual al derecho que al revés (ejemplo: "Anita lava la tina").
¿Qué practicas?: Manipulación de strings, limpieza de espacios y el concepto de "slicing" en Python ([::-1]).

# Opción C: Conversor de Números a Romanos (Lógica de Estructuras)
Este es un nivel un poco más alto de lógica pura.
El Reto: Pedir al usuario un número entero (del 1 al 10) y que el programa devuelva su equivalente en números romanos (I, II, III, IV...).
¿Qué practicas?: El uso de diccionarios o listas, que es la base de lo que luego usarás para simular o manejar datos de SQL/MariaDB
.
